## 📌 Sobre o Projeto
O **LGPD4DEVS** surge para preencher a lacuna entre a legislação (Direito) e a implementação técnica (Engenharia de Software). O foco principal é o **Artigo 14 da LGPD**, que trata da proteção de dados de crianças e adolescentes em ambientes digitais.

## 🚀 O Produto (Escopo)
Desenvolvimento de uma plataforma web que centraliza:
* ✅ **Checklist Interativo:** Avaliação de conformidade em tempo real.
* 📖 **Guia Prático:** Tradução da lei para linguagem técnica.
* 🛠️ **Ferramentas de Apoio:** Materiais para auxiliar na criação de plataformas seguras.

## 🎯 Objetivos
* **Geral:** Compreender as lacunas de segurança em ferramentas infantojuvenis e oferecer soluções práticas para profissionais de tecnologia.
* **Público-Alvo:** Desenvolvedores, engenheiros de software e gestores de TI.

## 💡 Justificativa
A segurança digital é a "faixa de pedestres" do mundo virtual. 
* **O Problema:** Falta de familiaridade dos DEVs com as práticas da LGPD e altos custos de implementação.
* **A Solução:** Mitigar riscos de crimes cibernéticos (malwares, phishing, cyberbullying) através do *Privacy by Design*.

## 🏗️ Especificações Técnicas
* **Ambiente:** Laragon (Apache)
* **Linguagem:** PHP 8.3
* **Banco de Dados:** MySQL 8.4
* **Estilização:** CSS3 Moderno (Layout Responsivo)

## 🌍 Alinhamento ODS (Agenda 2030)
Este projeto contribui para três pilares fundamentais:
1.  **ODS 4 (Educação de Qualidade):** Conscientização sobre proteção de dados.
2.  **ODS 9 (Inovação e Infraestrutura):** Fomento à inovação tecnológica responsável.
3.  **ODS 16 (Paz e Justiça):** Construção de ambientes digitais transparentes e seguros.
